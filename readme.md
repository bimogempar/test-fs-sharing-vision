HOW TO RUN!!!

- clone this repository

RUNNING BE
- create db mysql 'blog-post-crud'
- cd ./backend && go run main.go

RUNNING FE
- cd ./frontend
- npm install
- cp .env.example .env.local 
- masukkan setting environment app dan url pada .env.local
- npm start
